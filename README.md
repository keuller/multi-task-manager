## Project 

Simple webapp that demonstrantes multi project management.

## Description

[Fastify](https://www.fastify.io/) framework.

## Installation

```bash
$ npm install
```

## Running the server

```bash
# development
$ npm run start:dev

#production
$ npm start
```

## Running the UI

```bash
# watch mode
$ npm run start:ui

# build for production
$ npm run build:ui
```

## Test

```bash
# unit tests
$ npm run test

```
